🎬 Movie Hub
🚀 Features
🎥 Watch movies offline from local folders
📁 Scan and display movie catalog automatically
🗂️ Simple and organized movie library
💻 Modern desktop interface
⚡ Fast and lightweight performance
▶️ Easy movie playback
🛠️ Technologies Used
Java (core logic)
JavaFX (desktop UI framework)
FXML (UI structure)
CSS (styling and design)
📌 Project Idea

Movie Hub was created as a simple and modern desktop movie player for users who want quick access to their local movie collection. The application focuses on convenience, clean design, and smooth offline playback without requiring internet services.

🖥️ UI Concept

Although Movie Hub is a desktop application, its interface is designed to feel modern and user-friendly, inspired by streaming platforms and media libraries. It provides a clean navigation system and enjoyable viewing experience.
